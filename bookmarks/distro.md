# Linux OS

| [Fedora](https://fedoraproject.org/)          | Red Hat             | A cutting-edge, stable distribution that integrates the latest technologies.         | Developers and enthusiasts.                     |
| [Qubes](https://www.qubes-os.org/)            | Fedora (and others) | A security-oriented desktop OS that uses virtualization for apps.                    | High security and safety against malware.       |
